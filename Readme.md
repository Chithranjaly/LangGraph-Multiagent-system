# Multi-Agent Task Execution System (LangGraph)

## Overview
A controlled multi-agent system using:
- Planner → Executor → Critic loop
- Retry mechanism
- Structured task execution

## Features
- Step-by-step task decomposition
- Self-evaluation (PASS/FAIL)
- Retry with limits
- JSON output logging

## Run

```bash
python multi_agent_demo.py
